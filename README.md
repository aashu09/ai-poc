# POC
This is the AI Foundary POC app

---
>[!TIP]
> ##### **create virtual environment:**<br>
>    ```python -m venv env```
>    ###### **windows**: ```env\Script\activate.bat``` <br>
>    ###### **ubuntu**: ```source env/bin/activate```

>[!TIP] 
>    #### **For install dependency:**<br>
>    ```pip install -r requirements.txt```

>[!TIP]
>    #### **For migration:**<br>
>    ```alembic upgrade head```

>[!TIP]
>    #### **to run server:**<br>
>    ```uvicorn main:app```
